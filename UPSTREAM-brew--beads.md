# UPSTREAM-brew--beads

Tracking friction with [brew:beads](https://github.com/steveyegge/beads) (the `bd` CLI).

## Feature Requests

- **`bd init` should auto-gitignore `interactions.jsonl` and `.beads-credential-key`**
  (2026-03-28) — The `.beads/.gitignore` generated by `bd init` does not include
  `interactions.jsonl` (append-only audit trail) or `.beads-credential-key`
  (per-machine secret). Both files generate noise in git history or risk credential
  exposure. Verified: neither appears in this project's `.beads/.gitignore`.
  `bd doctor --check=gitignore` does not exist as a standalone flag — gitignore
  validation only runs as part of the default `bd doctor` check.
  Ownership: upstream · Workaround: full — manual gitignore + `git rm --cached`
